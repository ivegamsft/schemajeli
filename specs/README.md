# SchemaJeli Specifications

Organized specification documentation for the SchemaJeli metadata repository project.

## Directory Structure

- **api/** - API specifications and documentation
  - OpenAPI/REST API definitions
  - Endpoint documentation
  - Request/response schemas

- **architecture/** - System architecture and design
  - High-level architecture diagrams
  - Component design documents
  - Technology decisions (ADRs)

- **features/** - Feature specifications
  - Feature requirements and user stories
  - Functional specifications
  - Feature roadmap

- **data-model/** - Data model and database specifications
  - Database schema design
  - Entity relationship diagrams
  - Data dictionary

- **deployment/** - Deployment and infrastructure specifications
  - Infrastructure as Code (Terraform, etc.)
  - Deployment procedures
  - Configuration management
  - Azure setup and migration

- **security/** - Security specifications
  - Authentication and authorization specs
  - Data security requirements
  - Compliance requirements
  - Security architecture
